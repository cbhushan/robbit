Caratteristiche corrente

Il simulatore può prendere la pre-salvati dati (formato ASCII) per l'attuazione di simulazione in modalità non in linea, nonché il codice stesso di attuare online simulazione. Robbit ora supportano solo robot mobili. Per entrambe le situazioni offline e online, simulatore di fornire seguenti caratteristiche:
  1. Panaromic vista attraverso il controllo del mouse
  2. Pan in 3D
  3. Istantanea istantanea
  4. Play / Pause
  5. Animazione di controllo della velocit√†
  6. Controllo illuminazione
  7. Zoom In / Out
  8. A bordo di vista
  9. Immissione ostacolo (per la lettura dal file ASCII)
  10. Mozione sentieri
  11. Numerazione Robot
  12. Collision detection (attraverso le variazioni di colore)
  13. Arena di ridimensionamento
  14. Auto-rotazione vista
  15. Tempo di scorrimento durante la lettura dal file di log
