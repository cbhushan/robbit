Detailed Description

Class to store file data.

This record may be one from a log-file. It includes methods to build an index, to get based on the index, etc.

int File_Data::BuildFileIndex (  ) 

Builds index for quick reference within log file.

Stores the positions of get pointers to valid records in the log file, in a vector. This way, the file does not need to be traversed in serial fashion, and allows for quicker and random access to file data.
